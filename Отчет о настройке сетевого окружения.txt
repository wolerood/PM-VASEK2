Введение
Цель отчета: Документирование процесса настройки сетевого окружения для облачной виртуальной машины (VM) с операционной системой Ubuntu.

Шаги Настройки

Настройка Сетевых Параметров:

	Создана виртуальная частная сеть (VPC) с конфигурацией подсетей.
	Настроены группы безопасности для управления входящим и исходящим трафиком.
	Настройка IP-адресов:

	Назначен статический внешний IP-адрес для VM.
	онфигурирование внутренних IP-адресов в соответствии с требованиями подсети.
	Настройка DNS и Именных Сервисов:

	Настроен DNS для резолвинга доменных имен.
	Добавлены записи DNS для VM, если это необходимо.
	Настройка Сетевых Служб:

	Установлен и настроен SSH для удаленного доступа.
	Настроены дополнительные сетевые службы:
        (HTTP/HTTPS, FTP) в соответствии с потребностями проекта.

Тестирование Сетевых Соединений:

	Проведено тестирование соединения с VM через SSH.
